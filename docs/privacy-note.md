# Privacy Note

This repository is shared for portfolio purposes only.

The original Excel source files used in the project contain business-sensitive and personal information, therefore they are not included in this repository.

To protect privacy and confidentiality:

- The original dataset is **not included**
- Screenshots were **cleaned from sensitive information**
- Email addresses and personal data were **blurred**
- Some company names were **anonymized**

The repository focuses on demonstrating the **architecture, automation logic, and AI integration** rather than the raw data itself.
